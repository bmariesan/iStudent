# Project Details
## Prerequisites
- __[JDK 1.8](http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html)__

## How to build and run the project?
### Windows
```java
step 1:
gradlew clean build
step 2:
gradlew bootRun
step 3:
http://localhost:8080
step 4:
Enjoy coding :)
```
### Linux/Mac OS
```java
step 1:
./gradlew clean build
step 2:
./gradlew bootRun
step 3:
http://localhost:8080
step 4:
Enjoy coding :)
```
